# Employee Salary Analysis System

## Overview
Employee Salary Analysis System is a Python project developed using Pandas and Matplotlib.  
The project reads employee data from a CSV file, performs salary analysis, and visualizes insights using charts and graphs.

---

## Technologies Used
- Python
- Pandas
- Matplotlib

---

## Features
- Read employee data from CSV
- Display employee records
- Calculate average salary
- Find maximum salary
- Department-wise salary analysis
- Bar chart visualization
- Gender distribution pie chart
- Experience vs Salary scatter plot
